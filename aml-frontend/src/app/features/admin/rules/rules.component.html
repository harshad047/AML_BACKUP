Value Format Guide
  • AMOUNT: Numeric value (e.g., 10000)
  • COUNTRY_RISK: Risk score 0-10 (e.g., 7)
  • NLP_SCORE: Score 0.0-1.0 (e.g., 0.8)
  • KEYWORD_MATCH: Keyword text (e.g., cash, bitcoin)
  • PAST_TRANSACTIONS: lookbackDays|threshold (e.g., 30|10 or 7|50000)
  • VELOCITY: minAmount|minCount|windowHours|type (e.g., 100000|3|24|DEPOSIT)
  • STRUCTURING: maxSingle|maxSum|windowHours|types (e.g., 50000|300000|24|DEPOSIT)
  • BEHAVIORAL_DEVIATION: lookbackDays|percentile (e.g., 90|95)
  • AMOUNT_BALANCE_RATIO: Decimal ratio (e.g., 0.8 for 80%)
  • DAILY_TOTAL: threshold|windowHours|types (e.g., 500000|24|ANY)
  • NEW_COUNTERPARTY: lookbackDays|minAmount|types (e.g., 30|50000|TRANSFER)
  • PATTERN_DEPOSIT_WITHDRAW: requiredPairs|multiplier (e.g., 3|1.0)

Transaction Types: Use DEPOSIT, TRANSFER, WITHDRAWAL, ANY, or comma-separated (e.g., DEPOSIT,TRANSFER)
