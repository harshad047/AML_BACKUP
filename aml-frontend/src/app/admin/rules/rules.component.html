Value Format Guide
  • AMOUNT: Numeric value (e.g., 15000)
  • COUNTRY_RISK: Risk score 0-100 (e.g., 70)
  • NLP_SCORE: Score 0-100 (e.g., 80)
  • KEYWORD_MATCH: Keyword text (e.g., suspicious)
  • VELOCITY: Format: minAmount|minCount|windowHours|txType (e.g., 100000|3|24|DEPOSIT)
  • AMOUNT_BALANCE_RATIO: Decimal ratio (e.g., 0.8 for 80%)
  • NEW_COUNTERPARTY: Format: lookbackDays|minAmount|txTypes (e.g., 30|50000|TRANSFER)
  • DAILY_TOTAL: Format: maxAmount|windowHours|txTypes (e.g., 500000|24|DEPOSIT,TRANSFER)
  • PAST_TRANSACTIONS: Format: minCount|windowDays|minAmount|txType (e.g., 5|30|10000|TRANSFER)
